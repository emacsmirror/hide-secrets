* Hide secrets mode

Hide secrets mode is a minor mode for GNU Emacs to hide all kind of secrets in Emacs buffers,
using overlays.

At the moments it tries to hide:

 - user names
 - email addresses
 - passwords
 - IP addresses (both v4 and v6)
 - hash sums
 - public ssh keys

Please feel free to ask for additional secrets.
